# React-Vite-Template

## To init template use next commands:

### `yarn`
Installs packages

### `npm run prepare`
Enables pre-commit

### `chmod ug+x .husky/*`
### `chmod ug+x .git/hooks/*`
Only if you are Mac or Linux user: Gives rights to husky command

### `vite`
Starts dev server

### `yarn build`
To build prod version
